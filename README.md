# NewsSite

A full-stack web application for browsing, sharing, and interacting with news content. Built with ASP.NET Web API and vanilla JavaScript.

## What it does

- Browse news articles from external news APIs
- Ask AI-powered questions about news content
- Share articles with other users, like and bookmark content
- Report inappropriate content for admin review
- Admin panel for managing users, content moderation, and platform statistics

## Tech Stack

**Backend:** C#, ASP.NET Web API, SQL Server  
**Frontend:** JavaScript, HTML, CSS  
**APIs:** External News API, AI API integration

## How to Run

1. Make sure you have **.NET 7+** and **SQL Server** installed
2. Clone the project
3. Update your database connection string in `server/appsettings.Development.json`
4. Run the server
5. Open `client/Pages/index.html` in your browser
